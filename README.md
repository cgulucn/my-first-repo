# 我的第一次回购
> Life is always moving. 🚀

这是我的第一个 GitHub 项目，一个视频下载工具，支持无 AVC 格式下载，让视频保存更自由。

---

## ✨ 功能特点
- 📥 支持主流平台视频无水印下载
- 🎞️ 无 AVC 格式兼容，适配更多设备播放
- 🚀 操作简单，一键保存你喜欢的视频

---

## 📦 使用方法
1.  下载 `无AVC的.exe` 文件
2.  双击运行，按提示输入视频链接
3.  选择保存路径，等待下载完成即可
