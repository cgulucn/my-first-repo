# 我的第一次回购 | My First Repo
> Life is always moving. 🚀

这是一个视频下载工具，支持无 AVC 格式下载，让视频保存更自由。
This is a video downloader that supports AVC-free format, making video saving more flexible.

---

## ✨ 功能特点 | Features
- 📥 支持主流平台视频无水印下载 | Support watermark-free download from mainstream platforms
- 🎞️ 无 AVC 格式兼容，适配更多设备播放 | AVC-free format compatible with more devices
- 🚀 操作简单，一键保存你喜欢的视频 | Simple operation, save your favorite videos with one click

---

## 📦 使用方法 | Usage
1.  下载 `Any Video Converter.exe` 文件 | Download the `Any Video Converter.exe` file
2.  双击运行，按提示输入视频链接 | Double-click to run, enter the video link as prompted
3.  选择保存路径，等待下载完成即可 | Choose the save path and wait for the download to complete
